# PokeCraft

Server-side Pokemon plugin for Paper 1.21.x. No client mod required.
Bedrock/mobile players join via Geyser + Floodgate. Optional 3D pokemon models
are provided by BetterModel (free, open-source) for Java clients; Bedrock/mobile
players see each pokemon as a fitting vanilla mob.

See docs/ for architecture, setup and roadmap.
